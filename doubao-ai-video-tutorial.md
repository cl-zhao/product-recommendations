## 四、如何写视频Prompt（核心技巧）

### 4.1 Prompt公式

```
提示词 = 主体(主体描述) + 场景(场景描述) + 运动(运动描述) + 镜头语言 + 氛围词 + 风格化
```

### 4.2 镜头运动词汇库

| 镜头类型 | 英文 | 中文描述 |
|----------|------|----------|
| 推镜头 | zoom in | 镜头向前推进，特写聚焦 |
| 拉镜头 | zoom out | 镜头向后拉，全景展开 |
| 摇镜头 | pan | 镜头左右/上下摇动 |
| 移镜头 | tracking shot | 跟随主体移动 |
| 环绕 | orbit/around | 环绕主体旋转 |
| 跟随 | follow | 跟随目标移动 |
| 固定镜头 | static shot | 镜头固定不动 |
| 倾斜 | tilt | 镜头上下倾斜 |
| 希区柯克变焦 | Hitchcock zoom | 推进同时后退 |

### 4.3 运动幅度控制

**轻微运动关键词：**
- 轻微运动、轻轻、微微、缓缓、微颤

**剧烈运动关键词：**
- 快速、猛烈、激烈、迅疾、剧烈

### 4.4 视角类型

| 视角 | 说明 |
|------|------|
| 低角度 | 仰视，更具压迫感 |
| 高角度 | 俯视，展现全貌 |
| 正面 | 直接对话感 |
| 侧面 | 展示轮廓 |
| 鸟瞰 | 顶部俯视 |

### 4.5 文生视频 vs 图生视频

- **文生视频**：需要完整的6步公式（主体+场景+运动+镜头+氛围+风格）
- **图生视频**：重点描述动态变化和镜头动作，需要加"preserve composition and colors"保持构图色彩

## 六、AI视频生成的局限性

### 6.1 时长限制

| 主流AI工具 | 最大时长 | 说明 |
|------------|----------|------|
| 即梦/豆包 | 5-10秒 | 2026年主流水平 |
| 可灵 | 2分钟 | 国内较长 |
| Sora | 20秒 | 开放后支持 |
| Runway | 10-20秒 | 适度长度 |

**技术原因：**
- AI模型"嗷"一下直接生成全部帧，而非逐帧生成
- 计算资源限制，内存随时长指数增长
- 长视频会"跑偏"（分布偏移问题）

### 6.2 画质问题

- 高分辨率生成需要更长时间
- 细节可能在生成过程中丢失
- "时空错乱"问题：动作速度不符合真实物理

### 6.3 内容限制

- 人物一致性难以长时间保持
- 复杂物理交互（如碰撞、液体）仍有问题
- 文字生成准确度有限（中文尤其）
- 手部、面部细节经常失真

### 6.4 其他问题

- 生成速度（可灵较慢，即梦较快）
- 版权合规风险
- 音频生成质量参差不齐

## 七、视频生成的版权问题

### 7.1 版权归属现状

根据当前法律实践和平台政策：

| 情况 | 版权归属 | 说明 |
|------|----------|------|
| 纯AI生成 | 存在争议 | 捷克法院认定AI生成图像不受版权保护 |
| 深度修改（30%以上） | 用户 | 需有实质性人类智力投入 |
| 使用侵权素材训练 | 平台风险 | 用户需自行确保训练数据合法性 |

### 7.2 侵权风险类型

1. **生成内容侵权**：AI生成的视频可能模仿特定影视角色、风格
2. **训练数据侵权**：用版权作品训练AI模型是否合理使用尚未明确
3. **"魔改"视频**：使用AI"魔改"影视作品可能侵犯改编权

### 7.3 合规建议

✅ 保存创作日志（录屏、区块链存证）  
✅ 进行独创性修改（30%以上实质性改动）  
✅ 声明权利并登记版权  
✅ 避免使用侵权提示词（如具体明星名、影视角色名）  
✅ 关注数字水印、音色保护等新规动态  
✅ 优先加入平台合规计划（如YouTube AI音乐计划）

## 十、常见问题解答（FAQ）

### Q1：豆包和即梦AI有什么区别？

豆包是AI助手产品，视频生成是其中的一个功能；即梦是专业的一站式AI创作平台，视频和图像功能更全面。两者底层模型相同（Seedance）。

### Q2：AI生成的视频可以商用吗？

取决于具体平台条款和版权归属。建议进行30%以上的独创性修改，并保存创作证据。

### Q3：为什么生成的视频时长只有5-10秒？

这是当前技术限制。生成方式是"一次性"生成全部帧，时长受计算资源制约。可灵支持最长2分钟。

### Q4：如何保持视频中人物一致性？

使用Seedance 2.0的多主体一致性控制功能；或采用分镜生成+剪辑方式。

### Q5：生成的视频模糊怎么办？

选择更高分辨率（如720P）；优化Prompt增加"高清"、"4K"等关键词；检查首帧图片质量。

### Q6：AI视频生成需要什么配置的电脑？

网页端工具（豆包、即梦）无需高配；本地部署需要高显存GPU（建议16GB+）。
