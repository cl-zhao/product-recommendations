assistant: 这是 **Camoufox CLI** - 一个专为 AI 智能体设计的**反检测无头浏览器**工具

## 核心问题

AI 智能体用普通浏览器（Playwright、Puppeteer 等）浏览网页时，会被网站识别为机器人：
- `navigator.webdriver=true` 暴露自动化
- 无浏览器插件、统一的指纹特征
- 结果：验证码、登录墙、直接封禁

## 解决方案

**Camoufox** 是 Firefox 的修改版，在 C++ 引擎层面伪装：
- Canvas / WebGL / Audio 指纹伪装
- 真实的浏览器插件
- 随机化屏幕指标、字体列表
- `navigator.webdriver=false`

## CLI 特点

| 特性 | 说明 |
|------|------|
| **反检测** | C++ 级别指纹伪装，不是 JS 注入 |
| **省 Token** | 紧凑文本输出，专为 AI 优化 |
| **Ref 引用** | snapshot 返回 `@ref` 选择器，精确定位元素 |
| **多会话** | 支持多个隔离的浏览器实例 |
| **Cookie 管理** | 导入/导出 JSON，保持登录状态 |

## 基本用法

```bash
camoufox-cli open https://example.com   # 打开页面
camoufox-cli snapshot -i                 # 获取可交互元素
camoufox-cli click @e1                   # 点击元素
camoufox-cli fill @e3 "搜索内容"         # 填写输入框
camoufox-cli close                       # 关闭
```

## 架构

```
CLI ──Unix socket──▶ Daemon ──Playwright──▶ Camoufox(Firefox)
```
