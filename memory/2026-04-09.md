### CodexBar 项目研究（14:44 UTC+8）

**原始项目 steipete/codexbar**：macOS 专属（Swift/SwiftUI），使用 NSStatusItem/Keychain/WidgetKit/Sparkle 等 macOS 专属 API，**完全无法在 Windows 运行**。

**Windows 移植版 Finesssee/Win-CodexBar**：
- 技术栈：Rust + egui（跨平台 UI，非 WebView）
- 支持 16 个 AI Provider：Codex, Claude, Cursor, Gemini, Copilot, Antigravity, Windsurf, Zai, Kiro, Vertex AI, Augment, MiniMax, OpenCode Go, Kimi, Kimi K2, Perplexity
- 三种模式：原生 Windows 系统托盘 / WSL GUI（WSLg）/ 纯 CLI
- Cookie 读取：Windows DPAPI（Chrome/Edge/Brave）/ Firefox SQLite
- 发布页：https://github.com/Finesssee/Win-CodexBar/releases
- 安装包：CodexBar-<version>-x64-setup.exe
- ⚠️ 限制：WSL 中 Chromium Cookie 因 DPAPI 加密不可用，需手动 cookie

### JuliusBrussee/caveman 项目（15:26 UTC+8）

**项目性质**：Claude Code / Codex 插件，通过" caveman speak"压缩输出 token

**核心数据**：
- 输出 token 节省：平均 65%（Range: 22%-87%）
- 技术准确度：100%（研究支持，arxiv:2604.00025）
- 速度提升：约 3x

**四级压缩强度**：

| Level | 触发 | 效果 |
|-------|------|------|
| Lite | /caveman lite | 去填充词，保留语法 |
| Full | /caveman full | 默认，去冠词，断句 |
| Ultra | /caveman ultra | 最高压缩，全缩写 |

**文言文模式**（Wenyan）：

| Level | 触发 | 效果 |
|-------|------|------|
| Wenyan-Lite | /caveman wenyan-lite | 半文言，语法完整 |
| Wenyan-Full | /caveman wenyan | 全文言文 |
| Wenyan-Ultra | /caveman wenyan-ultra | 极简文言 |

**配套工具**：
- `/caveman-commit`：≤50 字符的 terse commit message
- `/caveman-review`：单行 PR review
- `/caveman:compress`：压缩 CLAUDE.md 等记忆文件，节省 ~45% input token

**安装方式**：
```bash
px skills add JuliusBrussee/caveman
claude plugin marketplace add JuliusBrussee/caveman
```

**重要说明**：
- 仅压缩输出 token，不影响思考/推理 token
- 文言文模式是"最节省 token 的人类书写语言"
- Windows Codex 用户需启用 `git config core.symlinks true`（需开发者模式或管理员）
- CI 有 eval 测试，token 节省可验证
